Features
--------
- **Easy to Use**: Designed with simplicity in mind
- **High Performance**: Fast and powerful
- **Asynchronous**: Non-blocking requests
- **Scalable**: Works with TDLib Server
- **Well-typed**: Clear methods and types
- **Plugin Support**: Extensible with plugins
- **Comprehensive**: Supports all TDLib types and functions
